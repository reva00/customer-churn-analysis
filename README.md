# Customer Churn Analysis

## 📌 Problem Statement
Analyze customer data to identify patterns and factors contributing to customer churn in a telecom company.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Key Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of churn patterns

## 🔍 Key Insights
- Customers with higher monthly charges are more likely to churn.
- Customers on month-to-month contracts have higher churn compared to long-term contracts.

## 📁 Dataset
Telco Customer Churn dataset (Kaggle)

## 📈 Conclusion
Pricing and contract type play a significant role in customer churn. Businesses can improve retention by focusing on pricing strategies and encouraging long-term contracts.# customer-churn-analysis
